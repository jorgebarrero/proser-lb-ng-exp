# InvService
Servicios


## Campos 
### inv_service_id
Numero de identificacion del registro del service

### inv_service_status
Estado del servicio
- A = Activo
- B = inactivo
 

### inv_service_chk
Marca de uso del campo 
- 0
- 1

### inv_service_type 
Tipo de servicio
- texto

### inv_service_name
Nombre del servicio
- texto 

### inv_service_shortname
Nombre corto del servicio 
- texto 

### inv_service_use
